
# Ontario Snow Day Predictor

Snow Day Predictor is a machine learning project that estimates the probability of school closures due to winter weather. Trained on weather data from Winter 2024, the model considers factors such as temperature, snowfall amounts, overnight accumulation, and wind speed, then outputs a probability score for upcoming weekdays.
